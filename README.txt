Para executar os testes são necessarias as seguintes ações:

Passo 1:
Intalar o Python 3.7 ou superior
Passo 2:
Após a instalação do Python instalar o Selenium
Numa janela de comando colar o texto "pip install selenium", a instalção será automatica
Passo 3:
Adicionar o GeckoDriver no caminho abaixo(drive incluso no repositorio git)
"C:\Users\'USUARIO'\AppData\Local\Programs\Python\Python"
Passo 4:
Adiconar o caminho "C:\Users\'USUARIO'\AppData\Local\Programs\Python\Python" ao PATH do sistema.
Passo 5:
Abrir o IDLE do Python
Passo 6:
Abrir o Arquivo Main.py
Passo 7:
Executar o arquivo(F5)
Passo 8:
O testes deve ser executado corretamente.
